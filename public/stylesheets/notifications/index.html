Basic Notifications

This is a set of 5 Basic Noficiation Boxes. These are all powered by jQuery which means you can click them to make them disappear.

LIGHTBULB: This is a lightbulb themed notification. This could be used to show uses useful information or tips related to the page they are on.

INFORMATION: This is an information message. This can be used to provide information to the users such notices about services.

WARNING: This is a warning message. This can be used for example on password pages to show the strength of a password..

SUCCESS: This is a success message. This is my favourite! This is usually shown to explain that an action the user took has been successful

FAILURE: This is a Failure message. Doom and gloom! This is usually reserved for form validation or perhaps if an action the user took has failed.

Bonus Notifications

Here are another 10 Bonus Notifications that you probably won't find in other sets.

BASKET: This is a message for the Basket. Could be used to display how many items the user has in their shopping basket

REMINDER: This is a Reminder message. You could use this on the homepage to remind users of some action that they need to take - e.g. renew their membership.

CONTACTS: This is a Contacts message. You could use this alongside some sort of contact list or list of members.

SETTINGS: This is a Settings message. You could easily use this to show settings/preferenaces that a user has selected.

COMMENT: This is a Settings message. You could easily use this to show settings/preferenaces that a user has selected.

DATE: This is a Date message. You could use this on a calendar page or simply just to tell the user what the date is.

DELETE: This is a Delete message. This is one that is often missing from notification packs, use instead of the Failure message.

EMAIL: This is a E-Mail message. You can use this to tell the user about their messages pending or unread items.

HELP: This is a Help Message. You could use this to help the user along the process of registering or doing something on your site.

USER: This is a User Message. You can use this to keep the user informed or direct messages to them specifically.
